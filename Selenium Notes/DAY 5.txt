-----------------------------------

import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;

public class Launch28
{
	public static void main(String[] args) throws InterruptedException
	{
		WebDriver driver = new ChromeDriver();
		driver.get("E:/Excelr_Selenium_Sep7/HTML/login.html");
		
		Thread.sleep(5000);
			
		WebElement un = driver.findElement(By.xpath("/html/body/input[1]"));
		un.sendKeys("sandesh");
		Thread.sleep(3000);
		
		WebElement pwd = driver.findElement(By.xpath("/html/body/input[2]"));
		pwd.sendKeys("12345");
		Thread.sleep(3000);
		driver.close();
		
	}
}

X-Path are classified into 2 types:
-----------------------------------
1. Absolute xpath
2. Relative xpath 

1. Absolute xpath:
------------------
	--> Absolute xpath are such xpath which start from root node[HTML]
	--> Absolute xpath are represented using '/'
	--> All the xpath discussed earlier will come under "Absolute xpath"
	--> Absolute xpath are complete xpath which starts from the root node and move through all the immediate child element

Types of Relative Xpath:
------------------------
1. R-Xpath using attributes
2. R-Xpath using Multiple attributes
3. R-Xpath using AND, OR operators

Write the xpath to match the email and password field of Facebook application

HTML code for email field:
--------------------------
<input type="text" class="inputtext _55r1 _6luy" name="email" id="email" data-testid="royal_email" placeholder="Email address or phone number" autofocus="1" aria-label="Email address or phone number">

HTML code for Password field:
-----------------------------
<input type="password" class="inputtext _55r1 _6luy _9npi" name="pass" id="pass" data-testid="royal_pass" placeholder="Password" aria-label="Password">
-------------------------------------------------------------------------------

1. R-Xpath using attributes:
----------------------------
Syntax:
-------
//tagname[@attr_name = 'Value']

Write the xpath to match the email field of facebook application

//input[@id='email']

------------------------------------------------------------------------------------

2. R-Xpath using Multiple attributes:
-------------------------------------
Syntax:
-------
//tagname[@attr_name = 'Value'][@attr_name = 'Value']

Write the xpath to match the email and password field of facebook application

email:
------
//input[@id='email'][@name='email']

password:
---------
//input[@id='pass'][@name='pass']

------------------------------------------------------------------------------------

3. R-Xpath using using AND, OR operators:
------------------------------------------
Syntax:
-------
//tagname[@attr_name = 'Value' and @attr_name = 'Value']
//tagname[@attr_name = 'Value' or @attr_name = 'Value']

Write the xpath to match the Email and password field of Facebook application

email:
------
//input[@id='email' and @name='email']
//input[@id='email' or @name='email']

password:
---------
//input[@id='pass' and @name='pass']
//input[@id='pass' or @name='pass']
