Version Control System:[VCS]/SCM/SCM
-----------------------
	--> Its a s/m which is responsible for managing changes in Projects i.e Multiple Versions of the Project will be maintained 

	--> Version Control System is also known as Software Configuration Management (SCM) OR 
Source Code Management (SCM) System

Why VCS ?
	
	Diagram 1
	
Need of Version Control System:
-------------------------------
	1. To maintain Multiple versions of the same Project
	2. At any given point of time we can have the backup code of the project
	3. we can differentiate multiple version of the same project
	4. we can run multiple versions of the same project simultaneously
	5. It helps us to keep the track about history of the Project

GIT [Global Information Tracker]:
---------------------------------
	Git is Distributed Version Control System Tool.
	--> Git is not acronym and hence no expansion.But most of the people abbreviated as 
 "Global Information Tracker".
	--> GIT is developed by Linus Torvalds(Finnish software engineer), who also developed 
Linux Kernel.
	--> Most of the companies like Microsoft,Facebook,Yahoo,LinkedIn,Intel using Git as 
Version Control System Tool.


Working with VCS(Git):
----------------------


1. Install Git  (Download Git for Windows)
2. Create a Separate Folder in Local Machine, paste the Selenium Project inside this folder
3. Initialise git
	git init
	
4. Check the status
	git status
5. add the project to git
	git add .
It will add the files to the current working directory

6.Commit the changes within git
	git commit -m "firstcommit"

firstcommit --> name
